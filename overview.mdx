## Quick start

To start using Momentic, get familiar with the following features:

<CardGroup num={2}>
  <ImageCard title="Test editor" img="https://mintlify.s3-us-west-1.amazonaws.com/momentic-preview/images/test-editor.jpeg">
    The test editor is how you will create and configure your tests. It is a simple, yet powerful, editor that allows you to create end-to-end tests interactively with no code.
  </ImageCard>
  <ImageCard title="Run viewer" img="https://mintlify.s3-us-west-1.amazonaws.com/momentic-preview/images/run-viewer.jpeg">
    The run viewer allows you to view any test run in the Momentic app. You can step through the test run and see the state of the UI at each step. It also provides insights into the test run, such as the duration and queue time.
  </ImageCard>
</CardGroup>
