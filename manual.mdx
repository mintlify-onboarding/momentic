title: Running a test manually
sidebarTitle: Manual
description: How to run a test manually with Momentic.
---

On the tests page, you can manually trigger a run for a test. This is useful if you want to run a test without waiting for the next scheduled run.

<Frame>![Running manually](/images/manually.gif)</Frame>

Clicking the run button will queue a run to be executed as soon as possible. In the results view, the run will be marked as manually triggered.
